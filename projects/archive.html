Summary

This portfolio experiment reframes a personal site as an editorial archive. The work is presented as case files surrounded by atmosphere, fragments, and small cinematic rituals.

Concept

The identity is intentionally quiet: grain, drawn worlds, hanging studies, a vinyl widget, and warm gallery light rather than loud portfolio spectacle.

Design / Build

The implementation stays GitHub Pages compatible, avoids heavy libraries, preserves semantic HTML, and uses CSS placeholders until final screenshots are added.

Reflection / Outcome

The result establishes a flexible archive identity that can grow with real projects while keeping the mood personal and cinematic.
